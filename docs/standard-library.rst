Standard Library
================

Currently implemented modules in :ref:`TL5 <syntax-tl5>` standard library:

* data structures (basic)
* time
* math (basic)
* os (basic)

Planned future modules in the standard library:

* zlib
* more data structures
* math (full)
* os (full)
* system
* IO
* multiprocessing
* paths and files
* more compressions
* web-server (DB, HTTP, ext...)
* GUI
